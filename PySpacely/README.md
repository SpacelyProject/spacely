# PySpacely

## What is in this folder?

This folder contains the core python code for Spacely.

Sub-folders:
- /logs/: log outputs generated by Spacely
- /output/: data file outputs generated by Spacely
- /requirements/: lists of required libraries
- /src/: source code for sub-components of Spacely
- /test/: pytest suite for Spacely

Python files:
- Master_Config.py: Parses Master_Config.txt to generate master configuration for Spacely.
- Spacely_Globals.py: Global variables
- Spacely_Utils.py: Spacely utility functions
- Spacely.py: Core code to launch Spacely

Shell scripts:
- Spacely.sh: Launch Spacely on Linux
- SetupLinux.sh: Install requirements for Spacely on Linux
- Spacely.ps1: Launch Spacely on Windows
- SetupWindows.ps1: Install requirements for Spacely on Windows

Files you will create:
- Master_Config.txt: Automatically created the first time you run Spacely; you will need to fill in the correct global config variables.
- /spacely-asic-config/: ASIC-specific routines. See documentation for more info.

## How do I get started?

Spacely is supported on Windows or Linux.

Refer to documentation on [Installing Spacely](https://github.com/SpacelyProject/spacely-docs/blob/main/fundamentals/Installing%20Spacely.md)
